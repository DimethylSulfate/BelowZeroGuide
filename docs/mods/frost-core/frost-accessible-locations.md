## [FROST - Accessible Locations](https://www.nexusmods.com/fallout4/users/51477666?tab=user+files)
:material-tag-plus: `Optional`

This is not a single mod, it is a collection of mods.
Opens up and changes locations that were previously locked in FROST. 


### Installation Instructions
<small>
:material-information-box:
**Note:** You need to download and install files from multiple mod pages.
</small>

1. Download and install **[Main Files - FROST - Accessible Fort Hagen :material-tray-arrow-down:](https://www.nexusmods.com/fallout4/mods/55398?tab=files)** with your mod manager
2. Download and install **[Main Files - FROST - Accessible Fort Strong :material-tray-arrow-down:](https://www.nexusmods.com/fallout4/mods/54873?tab=files)** with your mod manager
3. Download and install **[Main Files - FROST - Accessible Med-Tek Research :material-tray-arrow-down:](https://www.nexusmods.com/fallout4/mods/54564?tab=files)** with your mod manager
4. Download and install **[Main Files - FROST - Accessible ArcJet :material-tray-arrow-down:](https://www.nexusmods.com/fallout4/mods/53923?tab=files)** with your mod manager
